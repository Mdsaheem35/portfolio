parallel data processing

Build an parallel data processing workflow on AWS using Simple notification Service and Lambda Functions. In this project created a SNS topic and configured lambda functions as subscriber to the Topic. Each subscriber has filter policy assigned to it. whenever an message received by the topic it is only pulled by specific subscriber based on filter policy. This helps to reduce overall processing time and cost by allocating a specific job to specific subscriber rather than allowing all subscribes to process the job.
